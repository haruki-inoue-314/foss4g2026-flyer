# FOSS4G 2026 Hiroshima Flyer (Japanese)

This repository contains the source files for the FOSS4G 2026 Hiroshima flyer in Japanese. The flyer provides information about the FOSS4G conference, including dates, location, registration details, and contact information.

## Building the Flyer

Install [Typst](https://github.com/typst/typst) if you haven't already. Then, run the following command in the terminal to compile the flyer:

```shell
typst c foss4g-2026-flyer.ja.typ
```
